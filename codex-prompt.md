# Codex Build Prompt

Build a clean online gallery for the SKZOO LA Pop-Up project.

Use the uploaded image assets as the complete project library. Create a premium, dark, event-presentation style website where I can view the project images clearly.

Requirements:

1. Static site preferred: HTML, CSS, and JavaScript only.
2. Keep all images local in `assets/images`.
3. Use `assets/gallery-data.js` as the image manifest.
4. Layout sections:
   - Hero / project intro
   - Master ISO layout
   - Wide renders
   - Detail shots
   - Contact sheet / full library
5. Add a lightbox so I can click any image and view it large.
6. Add filters for All, Master Plan, Wide Renders, and Detail Shots.
7. Add download links for each full-resolution PNG.
8. Make the design feel high-end and presentation-ready: dark background, blue airport accents, clean typography, large image cards, smooth hover states.
9. Make it fully responsive for desktop, tablet, and mobile.
10. Do not crop images in the lightbox. Use object-fit: contain for full image review.
11. Do not rename or delete assets unless updating all references.

The goal is a review gallery that lets me quickly present and inspect all SKZOO LA pop-up render images online.
